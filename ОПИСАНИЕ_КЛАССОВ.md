•	Program – основной класс, содержащий метод Main, с него начинается запуск приложения.

o	public static void Main(string[] args) – метод запуска всей программы, точка входа в приложение;

•	App – класс главного приложения. Инициализирует XAML и создает главное окно с ViewModel.
